# Project Dossier

**Project Dossier** es una plataforma web de archivo documental donde se recopilan casos clasificados y desclasificados, así como teorías conspirativas **únicamente cuando están sustentadas por fuentes públicas y pruebas verificables**.

## Aviso legal importante
Este proyecto **NO afirma la veracidad de las teorías presentadas**.  
Todo el contenido:
- Se publica con fines **informativos, educativos y de archivo**.
- Se basa en documentos de dominio público, filtraciones ya publicadas por medios o fuentes oficiales.
- **No incluye información reservada, secretos de Estado no desclasificados ni datos personales sensibles**.
- Las interpretaciones son de los autores de cada ficha; Project Dossier no se hace responsable del uso que terceros hagan de esta información.

## Qué NO se permite publicar
- Datos personales de personas vivas (direcciones, DNI, teléfonos, correos, etc.).
- Material protegido por derechos de autor sin permiso expreso.
- Información clasificada que no haya sido oficialmente desclasificada.
- Contenido que incite a la violencia, al odio, al acoso o a actividades ilegales.

## Cómo contribuir
1. Clona el repositorio  
   `git clone https://github.com/tu-usuario/project-dossier.git`
2. Instala dependencias  
   `npm install`
3. Ejecuta el proyecto  
   `npm start`

Todas las aportaciones serán revisadas antes de publicarse.
